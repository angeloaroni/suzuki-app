# UI/UX Master Skill

Esta habilidad garantiza que cada elemento visual de SuzukiTracker se sienta profesional, moderno y "premium".

## 💎 Estética Premium

1. **Colores y Gradientes**: Usar paletas de colores suaves. Evitar colores primarios puros. Preferir tonos HSL balanceados.
2. **Glassmorphism**: Implementar efectos de desenfoque de fondo (`backdrop-blur`) en modales y barras de navegación.
3. **Sombras Profundas**: Usar sombras suaves y profundas para dar jerarquía visual.

## ✨ Animaciones y Feedback

1. **Framer Motion**: Cada transición debe ser suave. Usar `AnimatePresence` para entradas y salidas de elementos.
2. **Micro-interacciones**: Pequeñas animaciones al hacer hover en botones, tarjetas de canciones o al marcar la asistencia.
3. **Feedback Inmediato**: Mostrar esqueletos de carga (`Skeletons`) o indicators de carga discretos.

## 📱 Mobile-First Mindset

1. **Áreas de Toque**: Asegurar que todos los botones e inputs tengan un tamaño mínimo de 44x44px.
2. **Navegación Pulgar-Friendly**: Mantener las acciones principales al alcance del pulgar.
3. **Adaptabilidad**: Todo componente debe funcionar perfectamente desde un iPhone SE hasta un monitor ultrawide.
